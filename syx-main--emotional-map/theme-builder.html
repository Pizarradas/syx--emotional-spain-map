System Architecture Modules

SYX is a massive UI framework. Toggle these modular architectures to inject advanced semantic CSS into your export only when needed.

Fluid Typography & Spacing

Injects clamp() mathematical functions (--primitive-fluid-space-*, font-*). Perfect for responsive layouts that automatically scale across mobile and desktop without media queries.

Extended Color Palette

Generates and injects 5 complete 11-step color scales (Yellow, Red, Blue, Green, Purple) calculated from OKLCH math. Crucial for complex UIs, charts, or multiple semantic states.

Forms, Switches & Radios

Outputs hundreds of specific component variables (--component-form-*, --component-switch-*) required to perfectly render and align forms inside the SYX ecosystem.

Tables, Lists & Pagination

Injects structural tokens for complex data-grids, interactive components, active table states, and deep breadcrumb/pagination component UI alignments.

Disable Legacy Bridge

Opt-out of generating the backward compatibility layer for old SYX variables. Exports a strict, zero-bloat modern OKLCH token tree.
