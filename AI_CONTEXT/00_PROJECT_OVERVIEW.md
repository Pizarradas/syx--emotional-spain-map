
# Emotional Weather Map of Spain

## Project Summary
Interactive map of Spain that visualizes the dominant emotional tone of current news in different Spanish cities.

News is collected from Google News RSS, classified into emotions, aggregated per city, and displayed on a map.

Hovering a city marker changes the global UI theme using the SYX Design System.

## Core Metaphor
The system behaves like an emotional weather map.

Madrid → 😡 anger  
Valencia → 😢 sadness  
Bilbao → 😀 joy  

## Main Features
- automated news ingestion
- emotion classification
- aggregation by city
- interactive map visualization
- emotion-driven UI theming
- strict SYX design system usage
